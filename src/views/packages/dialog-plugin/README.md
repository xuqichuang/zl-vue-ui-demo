
### ZlDialogPlugin
> dialog弹框
###### 使用方法, 暂无默认值，推荐使用 ZlDialog引入

> 方法

| 名字                       | 参数             | 说明     | 版本要求 |
| -------------------------- | ---------------- | -------- | -------- |
| this.$zhenlv.dialog.show() | {background: ''} | 弹窗显示 | --       |
| this.$zhenlv.dialog.hide() | --               | 弹窗关闭 | --       |

> 参数说明

background: 弹框背景色
