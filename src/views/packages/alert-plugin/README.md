
### ZlAlertPlugin
> alert 提示框
###### 使用方法

> 方法

| 名字                      | 参数                            | 说明     | 版本要求 |
| ------------------------- | ------------------------------- | -------- | -------- |
| this.$zhenlv.alert.show() | {type:'',content:'',onHide(){}} | 弹窗显示 | --       |
| this.$zhenlv.alert.hide() | --                              | 弹窗关闭 | --       |

参数1 type: 标题

参数2 content: 内容

参数3 onHide(){}: 点击按钮时触发
