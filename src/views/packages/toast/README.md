
### ZlToastPlugin
> toast提示
###### 使用方法

> 方法

| 名字                      | 参数                                           | 说明      | 版本要求 |
| ------------------------- | ---------------------------------------------- | --------- | -------- |
| this.$zhenlv.toast.show() | {text:'提示文字', time: 2000, type: 'rhombus'} | toast显示 | --       |
| this.$zhenlv.toast.hide() | --                                             | toast关闭 | --       |

type 目前可选toast提示类型:  --, 可选 'rhombus'

text 目前可选文字:  自己定义的 text值, 无默认值

time 持续时间, 默认 2000
