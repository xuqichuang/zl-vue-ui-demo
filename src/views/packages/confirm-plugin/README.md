
### ZlConfirmPlugin
> confirm 提示框
###### 使用方法

> 方法

| 名字                        | 参数                                            | 说明     | 版本要求 |
| --------------------------- | ----------------------------------------------- | -------- | -------- |
| this.$zhenlv.confirm.show() | {type:'',content:'',onConfirm(){},onCancel(){}} | 弹窗显示 | --       |
| this.$zhenlv.confirm.hide() | --                                              | 弹窗关闭 | --       |

参数1 type: 标题

参数2 content: 内容

参数3 onConfirm(){}: 确认时触发

参数4 onCancel(){}: 取消时触发
